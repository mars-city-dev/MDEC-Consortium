# MDEC Global Metadata Standards v1.0.0

## Overview
The Meta Data Excellence Consortium (MDEC) establishes global standards for metadata excellence, eliminating "shitty metadata practices" worldwide.

## Core Standards

### Metadata Schema
All metadata must include the following required fields:
- id: GUID format
- name: Human-readable name
- path: File system path
- category: One of the approved categories
- created: ISO8601 timestamp
- modified: ISO8601 timestamp
- checksum: SHA256 hash

### Quality Requirements
- Minimum quality score: 80/100
- Excellent quality: 95/100
- Perfect quality: 100/100

## Certification Levels

### Bronze Certification
Requirements: Basic schema compliance, 50% quality score
Benefits: MDEC Bronze Certified badge, Basic validation tools

### Silver Certification
Requirements: Full schema compliance, 80% quality score, Automated validation
Benefits: MDEC Silver Certified badge, Advanced tools, Community support

### Gold Certification
Requirements: 100% quality score, AI automation, Self-healing systems
Benefits: MDEC Gold Certified badge, Full toolkit, Priority support, Certification authority
