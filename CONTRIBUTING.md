# Contributing

Thank you for contributing to Titaness Metadata Faktory and the MdEC Consortium.

## How to contribute
- Fork and create feature branches
- Open a PR describing intent and testing steps
- Lint and formatting are required (CI auto-checks will run)
- Do not change `CODEOWNERS` or `AGENTS.md` without explicit owner approval

## Governance
- Use `AGENTS.md` to configure automated agents and their scope
- Use `MAINTENANCE_PLAYBOOK.md` for release and maintenance policy
